# AI资料 V2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [x]`) syntax for tracking.

**Goal:** Ship a working AI资料 V2 vertical slice that preserves structured fields, stores layered memory, merges chapter patches deterministically, and keeps the existing UI usable.

**Architecture:** Keep the existing AI model configuration, routes, and Pinia store. Change backend AI memory persistence to transparent JSON so V1/V2 fields round-trip, then add frontend V2 types, migration, reconciliation, map/presentation adapters, and generation support. Preserve V1 display paths while letting new updates write `schemaVersion: 2`.

**Tech Stack:** Rust + axum + sqlx + serde_json; Vue 3 + Pinia + TypeScript + Vitest.
